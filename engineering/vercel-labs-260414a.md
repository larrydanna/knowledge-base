## TL;DR

- Vercel Labs is the R&D and incubation arm of Vercel, Inc. — focused on prototyping and open-sourcing next-generation web and AI developer tools
- Key projects include **v0** (AI UI generation), the **AI SDK** (streaming AI for JS/TS), and **Turborepo** (monorepo tooling)
- Vercel itself is the company behind **Next.js** and the Frontend Cloud platform — built for deploying and scaling modern web apps
- Founded by **Guillermo Rauch**; trusted by companies like Walmart, Netflix, and The Washington Post
- Mission: make high-performance, globally distributed web apps accessible to every developer

## What is Vercel?

Vercel, Inc. is a cloud platform company focused on the modern frontend — providing tools, hosting infrastructure, and a developer experience for shipping web applications at scale. Vercel created and maintains **Next.js**, the most widely used React-based framework for building static, server-rendered, and hybrid web apps.

The Vercel platform (often called the **Frontend Cloud**) offers:

- Instant global deployments via CDN edge network
- Serverless functions and edge middleware
- Preview deployments tied to pull requests
- Web analytics and performance monitoring
- First-class support for Next.js, Nuxt, Astro, SvelteKit, and more

---

## What is Vercel Labs?

Vercel Labs is the innovation and experimental branch within Vercel — their internal incubation team responsible for early-stage research, open-source tooling, and AI-forward developer products. Projects typically start in Labs and either graduate to full Vercel products or remain as open-source community tools.

Vercel Labs tends to release tools early to gather developer feedback and iterate rapidly.

## Key Projects from Vercel Labs

### v0

An AI-powered UI generation tool. You describe a component or page in plain language, and v0 generates production-ready React/JSX code using shadcn/ui and Tailwind CSS. Live at [v0.dev](https://v0.dev).

### AI SDK

An open-source TypeScript toolkit for building AI-powered applications with streaming support. Supports providers including OpenAI, Anthropic, Google, and more.

```bash
npm install ai
```

GitHub: [github.com/vercel/ai](https://github.com/vercel/ai)

### Turborepo

A high-performance monorepo build system (acquired by Vercel). Speeds up CI/CD pipelines by caching task outputs and parallelizing work across a monorepo.

```bash
npx create-turbo@latest
```

### Next.js (Core Product, Originated in Labs)

The full-stack React framework. Though now a flagship Vercel product, Next.js embodies the Labs ethos of experimenting and shipping fast: App Router, Server Components, and streaming all emerged from iterative research.

## Mission & Values

> *"Our mission is to enable developers to build and publish high-performant, beautiful apps and websites as easily as possible — across the globe, and for everyone."*

Core pillars:

- **Simplicity** — Zero-config deployments, instant previews, minimal setup
- **Performance** — Edge-first architecture, global CDN, serverless by default
- **Universality** — Works with any frontend framework, not just Next.js
- **Open Source** — Key projects (Next.js, AI SDK, Turborepo) are open-source and community-driven
