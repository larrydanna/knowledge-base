## Installation

To install Node.js, you can either download an installer from the official website or use a version manager. Version managers like nvm (Node Version Manager) are highly recommended because they allow you to switch between multiple Node versions easily and avoid permission issues when installing global packages. [1, 2] 
## Recommended: Using a Version Manager

* macOS and Linux (nvm):
1. Install nvm by running the install script from the [nvm GitHub repository](https://github.com/nvm-sh/nvm).
   2. Open a new terminal and run: nvm install --lts to get the latest Long-Term Support version.
   3. Verify by running node -v.
* Windows (nvm-windows):
1. Download the latest nvm-setup.exe from the [nvm-windows releases page](https://github.com/coreybutler/nvm-windows).
   2. Run the installer and follow the setup wizard.
   3. Open a command prompt and run: nvm install lts followed by nvm use lts. [2, 3, 4, 5, 6, 7, 8, 9] 

## Alternative: Using the Official Installer [10] 
If you prefer a standard setup, you can use the graphical installer:

   1. Visit the [official Node.js website](https://nodejs.org/en/download).
   2. Select the LTS (Long Term Support) version, which is recommended for most users for its stability.
   3. Download the installer for your system (Windows .msi, macOS .pkg, etc.).
   4. Run the installer and follow the prompts, keeping the default settings to ensure Node is added to your system's PATH.
   5. Verify the installation by opening your terminal or command prompt and typing:
   
   node -v
   npm -v
   
   This should display the version numbers for both Node.js and the Node Package Manager (npm). [3, 11, 12, 13, 14, 15, 16]
